# mihomo-config-builder v0.2.0

This release adds direct `nodes.txt` input so config generation no longer depends on subscription URL availability.

## Highlights

- New profile source field: `subscriptions[].nodesFile`
  - load local node URI lines directly (one line per node)
  - bypass subscription-converter URL dependency
- `nodes.txt` protocol support:
  - `ss://`
  - `vmess://`
  - `trojan://`
  - `vless://`
  - `hysteria2://` / `hy2://`
  - `socks5://` / `socks://` / `sock5://`
  - `http://` / `https://`
- Better multi-source merge behavior for duplicate `proxy-groups` names:
  - merges group members instead of dropping later groups
  - avoids losing nodes when multiple `nodesFile` inputs are combined

## Docs and examples

- Added `examples/nodes/nodes.txt`
- README updated with `nodesFile` usage and protocol list

# mihomo-config-builder v0.2.1

This patch release improves `nodes.txt` source parsing compatibility and subscription-link usability.

## Highlights

- `nodes.txt` now supports mixed lines:
  - direct node URI lines (existing behavior)
  - subscription URL lines (`http://` / `https://`) that are fetched and parsed automatically
- Subscription URL fragment suffix is supported for node naming, for example:
  - `https://example.com/sub?token=***#HK`
  - parsed nodes become `原名 HK`
- Node fragment names are decoded correctly when percent-encoded.

# mihomo-config-builder v0.2.4

This patch release improves subscription-derived proxy naming so region-identified nodes keep their country flag emoji after normalization.

## Highlights

- `subscriptions[].url` now preserves a region flag prefix when normalized proxy names are rewritten, for example `🇭🇰HK-k1-source`.
- `nodes.txt` subscription URL suffixes automatically gain a matching country flag when the suffix identifies a region, for example `#HK` becomes `🇭🇰HK`.
- Proxy-group references continue to be rewritten alongside renamed proxies, so normalized YAML subscriptions stay internally consistent.
